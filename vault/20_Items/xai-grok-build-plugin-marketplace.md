## Qué es
xAI lanzó el **Grok Build Plugin Marketplace** el 11 de junio de 2026, siguiendo el modelo
de extensiones de Claude Code skills y los plugins de Codex. Los plugins se instalan desde
el terminal de Grok Build y permiten al agente interactuar con servicios externos. Partners
en el lanzamiento: MongoDB, Vercel, Sentry, Chrome DevTools, Cloudflare y Superpowers.

El marketplace posiciona a Grok Build como competidor directo de Claude Code en el espacio
de agentes de coding con ecosistema de extensiones.

## Por qué importa (para mis proyectos)
Confirma la tendencia: todos los grandes proveedores están construyendo un ecosistema de
extensiones/plugins para sus agentes terminales. El modelo (instalar plugins desde CLI,
acceso a herramientas de infra) es el mismo en Claude Code, Codex y ahora Grok Build.
Para proyectos que evalúan harness, importa saber qué ecosistema de plugins/skills tiene
más profundidad y calidad — hoy Claude Code va en cabeza, pero xAI está cerrando la brecha.

## Cómo se usa / requisitos
Disponible para usuarios de Grok Build (plan xAI). Instalación via terminal de Grok.
Los plugins actuales cubren: bases de datos (MongoDB), deploy (Vercel), monitoreo (Sentry),
debugging web (Chrome DevTools), edge/CDN (Cloudflare) y AI infra (Superpowers).
