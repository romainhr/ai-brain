## Qué es
El 2026-06-02 OpenAI lanzó **plugins de Codex**: bundles que combinan skills, integraciones de
aplicaciones y configuraciones de MCP server en un solo paquete instalable.

## Por qué importa (para mis proyectos)
Señala convergencia del ecosistema hacia "paquetes de capacidades" (skills + MCP + integraciones).
Relevante si se evalúa Codex como alternativa/complemento, o como inspiración para empaquetar
flujos propios.

## Cómo se usa / requisitos
Dentro del ecosistema Codex de OpenAI. Verificar disponibilidad y formato exacto del plugin en
la documentación oficial de OpenAI.
