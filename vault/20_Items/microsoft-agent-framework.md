## Qué es
Framework para construir agentes presentado/ampliado en Microsoft BUILD 2026 (2 jun), orientado a
orquestación y despliegue en el ecosistema Microsoft/Azure.

## Por qué importa (para mis proyectos)
Otro contendiente en el espacio de SDKs de agentes (junto a Claude Agent SDK, OpenAI Responses,
LangChain). No es prioritario para mi stack actual; queda como referencia del panorama.

## Cómo se usa / requisitos
- SDK de Microsoft; integración pensada para Azure AI / .NET y Python.
