## Qué es
GPT-5.5 y la variante **Instant** (9 jun 2026): mejoras de personalización y respuestas más
claras/concisas, ya como modelo por defecto de ChatGPT Go/Free. El selector de modelos se
simplificó en tiers Instant / Medium / High / Pro.

## Por qué importa (para mis proyectos)
Marca el listón del competidor directo de Claude. No cambia mi stack (centrado en Claude +
modelos free), pero conviene seguirlo para decisiones de modelo y para comparar precio/calidad.

## Cómo se usa / requisitos
- API de OpenAI (`OPENAI_API_KEY`); también accesible vía graphify backend `openai`.
