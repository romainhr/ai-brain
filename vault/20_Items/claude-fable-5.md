## Qué es
Modelo liberado por Anthropic el 2026-06-09 junto a Mythos 5. Fable 5 es la variante de uso
general más segura, con mejoras en coding, trabajo de conocimiento, visión, memoria y
long-context.

## Por qué importa (para mis proyectos)
Es el candidato a modelo base para asistentes de código y flujos agénticos donde importa la
calidad y el contexto largo. Si un proyecto usa una versión anterior de Claude, evaluar migrar.

## Cómo se usa / requisitos
Disponible vía API de Anthropic y en Claude Code. Verificar el ID de modelo y pricing en la
documentación oficial antes de fijarlo en producción.
